# scrabble_score
Takes in a word from the user, calculates the correct Scrabble score and displays the results.

###Gems used
```erb
sinatra
sinatra-contrib
```

###Install Bundler
```erb
$ gem install bundler
```

###Run Bundler
```erb
$ bundle
```
